# 🏭 WMS - Warehouse Management System  
### Sistem Manajemen Gudang Berbasis Python

Aplikasi ini digunakan untuk mengelola data barang, stok, serta operasional warehouse secara sederhana, rapi, dan terstruktur menggunakan Python.

---

# ✨ Features

- 📦 Manajemen data barang gudang (tambah, ubah, hapus, lihat)
- 📊 Pengelolaan stok masuk & keluar
- 🔍 Pencarian data barang
- 📁 Penyimpanan data yang terstruktur
- ⚡ Interface sederhana dan mudah digunakan

---

# ⚙️ Requirements

- Python 3.x  
- pip (Python package manager)

---

# 📥 Installation & Setup

Ikuti langkah berikut untuk menjalankan project:

```bash
# 1. Clone repository
git clone https://github.com/Hisyam-L/WMS-Warehouse-Management-System-.git
cd wms

# 2. Buat virtual environment

# Windows
python -m venv venv
venv\Scripts\activate

# Linux / MacOS
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Jalankan project
python main.py
